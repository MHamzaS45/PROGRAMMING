What these exercises focus on
- if / elif / else branching to handle multiple cases

- Using break and continue appropriately
- Simple validation of user input

How to run
- Run each script with Python, e.g.: python "Week2/A2 Task 7.py"
- Test edge cases (empty input, boundary values) to confirm logic handles them.

Notes and learning goals
- Understand logical operators (and, or, not) and short-circuit evaluation.
- Practice writing clear, well-indented conditional blocks and loop bodies.
- Aim to keep functions small and focused if tasks include helper functions.
